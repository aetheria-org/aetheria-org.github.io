About

What is Aetheria?

Aetheria (ASM) is a Forge mod for Fakepixel Skyblock that adds QoL features that make the game more enjoyable — custom scoreboards, Diana waypoints, dungeon timers, HUD overlays, and more.

Being open source and community-driven, the code is public, contributions are welcome, and nothing runs in the background that you can't see for yourself. ASM is licensed under the GNU Lesser General Public License v3.

Installation
01

Download the mod

Grab the latest .jar from Modrinth or the official discord server's releases channel.

02

Find your game folder

In your launcher, look for a Game Directory, Open Folder, or Instance Folder button in your profile settings.

03

Add the mod

Open the mods folder inside your game directory — create it if it doesn't exist. Move the .jar file in there.

04

Launch with Forge

Select your Forge 1.8.9 profile and hit play. Don't have Forge? Most launchers have a built-in installer. If not, grab it at files.minecraftforge.net.
